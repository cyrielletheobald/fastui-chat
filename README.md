# fastui-chat

A minimalistic ChatBot Interface in pure python.

## How to use

```bash
> ./dev-install.sh
```

Si cette commande ne fonctionne pas, faire :
```bash
> echo 'export PATH="$HOME/.rye/shims:$PATH"' >> ~/.bashrc
> source ~/.bashrc 
> rye sync
> rye shell
```

Puis,
```bash
> startapp
```
OU 
```bash
> python src/fastui_chat/app.py
```
